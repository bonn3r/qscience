Créditos e Agradecimentos

HTML5 UP

Template original: “Editorial” por @ajlkn. Licenciado sob CCA 3.0. O design base do site foi construído a partir deste template, com adaptações e personalizações realizadas por mim.

Font Awesome

Ícones vetoriais: Utilizados para ilustrar os itens do menu, as áreas de interesse e os ícones de contato. A versão gratuita (6.5.2) foi incorporada via CDN.

MathJax

Renderização de notação matemática: Permite a exibição de equações LaTeX nas páginas, essencial para um site de ciências. A versão 3 é carregada via CDN.

CERN Document Server

Imagem do detector CMS: A fotografia do primeiro cilindro interno do rastreador do CMS (Tracker Inner Barrel) foi obtida no repositório do CERN. Créditos: M. Brice, 2006.

Resenha Sci‑Fi

Inspiração: O site do professor Derley foi a principal motivação para criar este espaço. A estrutura de divulgação científica e a curadoria de conteúdo serviram de modelo para o Q‑Science.

Logotipos e ícones próprios

Os logotipos das redes sociais e o ícone do gato (cat-logo.svg) foram criados ou editados por mim (Jimi.7z) para uso exclusivo neste site.

Caso algum crédito tenha sido omitido, por favor entre em contato para que eu possa corrigir.
